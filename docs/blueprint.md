# **App Name**: DoAll

## Core Features:

- Task Input: Simple input field for users to enter tasks.
- Task Display: Display tasks in a clear, readable list format.
- Task Completion: Allow users to mark tasks as complete.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) for calmness and focus.
- Background color: Very light blue (#F0F8FF) for a clean, unobtrusive look.
- Accent color: Pale purple (#BDB2FF) to add subtle visual interest to task interactions.
- Body and headline font: 'PT Sans' for a clear, modern, easy-to-read style.
- Use simple, universally recognized icons for task status and actions.
- Use minimal animations (e.g., a simple fade or slide) when tasks are added or completed, providing subtle feedback to the user.
